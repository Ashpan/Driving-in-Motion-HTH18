Welcome to Driving in Motion!

We are team of talanted developers who made a driving practice tool using a Leap Motion, Unity and Java

Check it Out!

What is this?!??!?!?!

We wanted to test peoples preparedness in a stressful car incidents. In order to do this we decided to use a Leap Motion to detect your hands to emulate a steering wheel. Then using Unity, we created a map and a car simulator, then we created scenarios (such as brake failure, icy patches, etc) to test a persons preparedness.
